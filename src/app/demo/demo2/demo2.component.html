Demo 2 : Les pipes

Propre à angular, ils permettent de formatter les données à l'afichage.

Les pipes déjà existants :

Les pipes string

Sans pipe : {{bonjour}}

En majuscule : {{bonjour | uppercase }}

En minuscule : {{bonjour | lowercase }}

En title case : {{bonjour | titlecase }}

Les pipes number

Un menu burger cheese :

Sans pipe : {{prix}}

Avec pipe pour arrondir : {{prix | number : '0.0-2'}}

Les pipes de monnaie

Avec pipe de monnaie : {{ prix | currency }}

Avec pipe de monnaie : {{ prix | currency : 'EUR' }}

Les pipes de date

Sans pipe : {{ajd}}

Avec pipe (sans param) : {{ajd | date}}

Avec pipe (avec param basique): {{ajd | date : 'dd/MM/yyyy'}}

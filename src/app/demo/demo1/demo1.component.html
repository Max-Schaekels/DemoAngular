Demo 1 : Les bindings

Le binding c'est le concept de pouvoir relier la partie code (TS) avec la partie HTLM

Property Binding

Permet de relier une propriété présente dans notre composant avec notre HTML

One-Way Binding

Permet d'afficher une propriété dans la page (celui que vous allez le plus utiliser)

Exemple : {{ prenom }}
